## Installation
All requirements should be detailed in requirements.txt. Using virtual environment is recommended.
```
    (optional) virtualenv .
    (optional) source bin/activate
    pip install -r requirements.txt
```

## Run the code
```
    python AveragedDQN.py --gpu_no=5 --num_model=5
    python DQN.py
```
